# **App Name**: WEBz Solutions

## Core Features:

- Dynamic Home Page: Responsive hero section, 'Why WEBz?' highlights, key industry cards, and a clear 'How It Works' process to introduce WEBz's mission.
- About Us Showcase: Detailed company history, founder introduction, and core values presented to build trust and explain the company's vision.
- Service Listing: Structured service cards detailing AI/ML solutions, analytics, custom apps, and process optimization, each with a request option.
- Project Case Studies: Case study placeholders demonstrating past project successes, including industry, problem, solution, and impact.
- Contact Form and Information: A clean contact form to capture inquiries and display key contact details (email, phone, city) for WEBz.
- Intelligent Inquiry Routing: Uses an AI tool to classify the problem descriptions submitted through the contact form and route them to the appropriate team or specialist, using the industry specified in the form as context.
- Firestore Integration: Capture client requests for services into a Firestore database, and present the database to administrators.

## Style Guidelines:

- Primary color: Teal (#008080) for a modern, tech-focused feel reminiscent of engineering solutions.
- Background color: Light grayish-blue (#E0F2F7), providing a clean, neutral backdrop.
- Accent color: Royal blue (#4169E1) to highlight calls-to-action and key interactive elements.
- Body and headline font: 'Inter', a sans-serif font for a clean and modern appearance. Code snippets (if any) will also be rendered in 'Inter'.
- Simple, modern icons that reflect AI, machine learning, and engineering concepts.
- Clean and structured layout with clear section headings, designed for easy content updates and scalability.
- Subtle hover effects on buttons and cards to enhance user interaction without overwhelming the interface.
